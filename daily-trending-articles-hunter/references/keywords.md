# 领域关键词配置说明

## 关键词的作用

关键词配置用于：
1. **匹配文章领域**：根据标题中出现的关键词，将文章分类到对应领域
2. **计算热度分数**：关键词匹配度是热度计算的指标之一
3. **筛选相关内容**：过滤掉与目标领域无关的文章

## 自定义关键词

### 添加新领域

在 `config.json` 中添加新的领域和关键词：

```json
{
  "keywords": {
    "新领域名称": ["关键词1", "关键词2", "关键词3"]
  }
}
```

### 调整关键词权重

热度计算中，关键词匹配度占比20%。如果想让某些关键词权重更高：

1. 在 `calculate_trending.py` 中修改 `calculate_keyword_score` 方法
2. 为不同关键词设置不同的权重值

## 关键词匹配规则

1. **不区分大小写**：Claude 和 claude 都能匹配
2. **部分匹配**：标题包含关键词即可匹配
3. **多关键词累加**：匹配越多关键词，分数越高
4. **跨领域匹配**：一篇文章可能匹配多个领域

## 热度计算公式

```
关键词匹配分数 = 匹配关键词数 / 总关键词数 × 0.3

综合热度 = (
    归一化点赞数 × 0.3 +
    归一化浏览量 × 0.2 +
    归一化评论数 × 0.2 +
    关键词匹配分数 × 0.2 +
    时间衰减因子 × 0.1
) × 平台权重
```

## 更新建议

根据你的关注领域，建议：

1. **定期更新关键词**：添加新兴技术和工具
2. **调整领域分类**：根据需求合并或拆分领域
3. **优化关键词列表**：移除过于宽泛的关键词，添加更精准的词汇

## 示例

### 示例1：AI文章匹配

标题：`"Claude Code发布：AI编程新纪元"`

匹配结果：
- "Claude" → AI工具
- "AI编程" → AI工具
- 匹配关键词：["AI工具:Claude", "AI工具:AI编程"]
- 领域分类：["AI工具"]

### 示例2：跨领域文章

标题：`"用Vercel部署Next.js实现的AI写作工具"`

匹配结果：
- "Vercel" → 网页开发
- "Next.js" → 网页开发
- "AI写作" → AI工具
- 匹配关键词：["网页开发:Vercel", "网页开发:Next.js", "AI工具:AI写作"]
- 领域分类：["网页开发", "AI工具"]
